One sentence definition

ImagePullBackOff means kubelet tried to pull the container image, failed, and is backing off before retrying.

What It Means

  • Kubernetes accepted the Pod YAML.
  • The scheduler placed the Pod on a node.
  • The container runtime could not pull the image.

Common Causes

  • Typo in image name.
  • Missing tag.
  • Private registry credentials missing.
  • Network or DNS issue on the node.

Fix Flow

kubectl describe pod nginx
kubectl get events --sort-by=.lastTimestamp
kubectl set image pod/nginx nginx=nginx:1.25

Imperative Quick Commands for CKA Exam

kubectl run nginx --image=nginx:badtag
kubectl describe pod nginx
kubectl get events --sort-by=.lastTimestamp
kubectl delete pod nginx
kubectl run nginx --image=nginx:1.25
The Mantra "Generate clean YAML, verify with kubectl, and keep the note short enough to revise."
