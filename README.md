# CKA Revision Notes

A lightweight static HTML study hub for CKA revision.

## Links

- Live site: https://programmingtothecore.github.io/cka-revision-notes/
- GitHub repo: https://github.com/programmingtothecore/cka-revision-notes

## Open Locally

Open `index.html` in a browser.

Key pages:

- `index.html` - visual hub
- `cka-topic-index.html` - searchable topic index
- `cka-revision-flow.html` - sequential revision path
- `cka-imperative-commands-cheatsheet.html` - one-glance CKA commands
- `cka-add-custom-page.html` - custom note generator

## Hosting

This is a plain static site. It can be hosted with:

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel static deployment

No build step is required.

## Notes

Do not commit real kubeconfigs, certificates, private keys, or cluster secrets. The repository is intended for study notes and example manifests only.
